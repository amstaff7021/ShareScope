# ShareScope

Upload and share any file in two clicks, including videos too large for Discord.

## Installation

```powershell
python -m pip install -r requirements.txt
```

## Lancement

```powershell
python app.py
```

## Utilisation

- ouvre l'app
- glisse un fichier dedans ou clique sur `Browse files`
- recupere le lien puis partage-le

## Historique

- `Copy` copie le lien
- `Open` ouvre le lien
- le survol affiche la taille, la date de creation et l'expiration
